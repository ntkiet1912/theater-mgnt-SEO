## ğŸ“ Learning Points

### Key Concepts

1. **Ngrok = Public URL for Localhost**
   - Localhost is private (only your machine)
   - Ngrok creates public URL
   - Tunnel forwards requests

2. **CORS = Browser Security**
   - Frontend (Vercel) â†’ Different domain from Backend (Ngrok)
   - Browser blocks cross-origin requests
   - Backend must explicitly allow Vercel domain

3. **Environment Variables = Configuration**
   - Same code, different URLs
   - Development: localhost
   - Production: ngrok/deployed backend

4. **Free Plan Limitations**
   - URL changes each restart
   - "Visit Site" warning page
   - Acceptable for testing, not production
